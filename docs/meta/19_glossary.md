# 19. 术语表

## Paoji

开放式表情对象协议的项目名。

## YS Token

可复制、可传播、可校验的短码入口。

## Visual Hint

Token 中给人看的视觉提示区。不是权威数据。

## Payload

Token 中给机器解码的数据区。

## PJD

Paoji Document，表情的明文结构。

## PJP

Paoji Pack，组件包 / 资源包。

## PJR

Paoji Registry，注册表。

## Resolver

负责把 Token 展开成 PJD 的解析器。

## Renderer

负责把 PJD 渲染成图像的渲染器。

## Component

可复用、可参数化、可降级的表情组件。

## Slot

组件被放置的位置语义，例如 eyes、mouth、deco。

## Anchor

组件对齐点，例如 leftEye、rightEye、mouth。

## Fallback

降级显示方案。

## Trust Level

组件或资源包的信任等级。

## Capability

渲染器或组件需要的能力，例如 `svg.gradient`、`motion.basic`。

## Namespace

全局命名空间，用于避免组件 ID 冲突。
