# 15. 设计师检查清单

> English version: [../en/components/15_designer_checklist.md](../en/components/15_designer_checklist.md)

用于设计师提交 Paoji 组件或组件包前自检。

## 1. 组件基础

- [ ] 使用 1024×1024 viewBox。
- [ ] 组件类型明确。
- [ ] 命名空间完整。
- [ ] 有版本号。
- [ ] 有显示名和描述。
- [ ] 有中文说明文本。
- [ ] 有英文说明文本。
- [ ] 有 license。

## 2. 可组合性

- [ ] 有合理锚点。
- [ ] 能与官方 Core 组件组合。
- [ ] 参数不依赖固定背景。
- [ ] 小尺寸仍可辨认。
- [ ] 不和其他标准 slot 冲突。

## 3. 参数

- [ ] 参数有类型。
- [ ] 参数有默认值。
- [ ] 数值参数有 min / max。
- [ ] 颜色参数可继承 palette。
- [ ] 参数数量不过多。

## 4. Fallback

- [ ] 有同类官方 fallback。
- [ ] 有 Visual Hint。
- [ ] 有 alt 文本。
- [ ] 在缺少 pack 时仍可降级显示。

## 5. 安全

- [ ] 不包含 script。
- [ ] 不包含事件处理器。
- [ ] 不依赖外部图片。
- [ ] 不依赖远程字体。
- [ ] SVG 通过安全扫描。
- [ ] 文件大小合理。

## 6. 视觉质量

- [ ] 32px 尺寸可辨识。
- [ ] 64px 尺寸有细节。
- [ ] 256px 尺寸美观。
- [ ] 1024px 尺寸不粗糙。
- [ ] 透明背景可用。
- [ ] 深浅背景都可读。

## 7. 发布

- [ ] manifest 完整。
- [ ] preview.png 已生成。
- [ ] hash 已生成。
- [ ] signature 可选但推荐。
- [ ] Registry 元数据完整。
