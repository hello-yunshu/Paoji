# 19. Glossary

> 中文版：[../../meta/19_glossary.md](../../meta/19_glossary.md)

## Paoji

The project name of the Open Expression Object Protocol.

## YS Token

A short-code entry that is copyable, transmissible, and verifiable.

## Visual Hint

The human-readable visual hint area in a Token. Not authoritative data.

## Payload

The machine-decodable data area in a Token.

## PJD

Paoji Document, the plaintext structure of an expression.

## PJP

Paoji Pack, a component pack / resource pack.

## PJR

Paoji Registry, the registry.

## Resolver

The parser responsible for expanding a Token into a PJD.

## Renderer

The renderer responsible for rendering a PJD into an image.

## Component

A reusable, parameterizable, degradable expression component.

## Slot

The positional semantics where a component is placed, e.g. eyes, mouth, deco.

## Anchor

A component alignment point, e.g. leftEye, rightEye, mouth.

## Fallback

A degraded display scheme.

## Trust Level

The trust level of a component or resource pack.

## Capability

A capability required by a renderer or component, e.g. `svg.gradient`, `motion.basic`.

## Namespace

A global namespace used to avoid component ID conflicts.
