# Paoji Brand Assets

> 中文版：[README.md](README.md)

This directory contains the project-level brand assets for Paoji. Official expression components live in [`../../packs/paoji-core/README.en.md`](../../packs/paoji-core/README.en.md). Asset descriptions support both Chinese and English.

| Asset | Usage | Format |
|---|---|---|
| `brand/paoji-mark.svg` | Avatar, favicon, square icon | SVG |
| `brand/paoji-logo.png` | README and documentation header | PNG, 620 × 373 |
| `brand/paoji-hero.png` | README hero and social preview | PNG, 1280 × 640 |
| `brand/protocol-flow.svg` | Protocol flow overview | SVG |

## Usage Guidelines

- Keep clear space of at least `1/4` of the mark width.
- Prefer the original artwork on light backgrounds; do not rearrange its modules.
- Asset descriptions support both Chinese and English; each language version should be kept in sync and complete.
- Third-party packs may use their own visual identity but must not imply official Paoji status.
