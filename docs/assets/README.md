# Paoji 品牌资源 / Brand Assets

> English version: [README.en.md](README.en.md)

这里集中保存 Paoji 的项目级品牌视觉。官方表情组件位于 [`../../packs/paoji-core/`](../../packs/paoji-core/README.md)。资源说明支持中英双语。

This directory contains the project-level brand assets for Paoji. Official expression components live in [`../../packs/paoji-core/`](../../packs/paoji-core/README.md). Asset descriptions support both Chinese and English.

| 资源 / Asset | 用途 / Usage | 格式 / Format |
|---|---|---|
| `brand/paoji-mark.svg` | 头像、favicon、方形图标 / Avatar, favicon, square icon | SVG |
| `brand/paoji-logo.png` | README、文档页眉 / README and documentation header | PNG, 620 × 373 |
| `brand/paoji-hero.png` | README 主视觉、GitHub Social Preview / README hero and social preview | PNG, 1280 × 640 |
| `brand/protocol-flow.svg` | 协议流程说明 / Protocol flow overview | SVG |

## 使用原则 / Usage Guidelines

- 标志周围至少保留图标宽度 `1/4` 的留白。
- 浅色背景优先使用原始版本；不要改变各模块的相对位置。
- 资产说明支持中英双语；各语言版本应保持同步与完整。
- 第三方组件包可以建立自己的视觉语言，但不应伪装成 Paoji 官方资源。

- Keep clear space of at least `1/4` of the mark width.
- Prefer the original artwork on light backgrounds; do not rearrange its modules.
- Asset descriptions support both Chinese and English; each language version should be kept in sync and complete.
- Third-party packs may use their own visual identity but must not imply official Paoji status.
